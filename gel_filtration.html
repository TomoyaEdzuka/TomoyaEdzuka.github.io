Gel filtration

  1. Column name: Superdex 200 Increase 10/300 GL
  2. Bed volume: 24 mL
  3. Sample volume 25 µL ~ 500 µL
  4. Limit pressure: 435 psi
  5. Max flow rate (water, low temperature): 0.9 ml/min
  6. Max flow rate (20 % ethanoal low temperature): 0.45 ml/min

  • GE Healthcare home page (Superdex column)
  • Biorad chromatography system documentation

Preparation

  1. Filtrate buffer, sonicate to break small bubbles
  2. Install "Backpressure Regulator" to the chromatography (the Backpressure Regulator helps eliminate bubble formation within the detector. without this, flow rate can be unstable). Don't forget to uninstall after use.

    Connect the backpressureregulator after the Conductivity monitor in the system plumbing.

  3. Wash column with DW
  4. Equilibrate column with buffer
  5. Filtrate or ultra-centrifuge your sample

Column wash

  1. Change flow-path to 1-1
  2. Change the mode to P (Purge)
  3. Put A and/or B tubes in water
  4. Replace pump content with DW
    1. Insert 20 mL syringe to the priming port.
    2. Turn the port counterclockwise.
    3. Pull slowly 15 mL with the syringe.
  5. Push "Purge" A and/or B button for ~ 5 s
  6. Run wash protocol (column volume x 3 (72 mL))
  7. Repeat the above steps with your buffer
  8. After the experiment, wash column with DW and replace with 20 % ethanol for long storage. If you're going to do the experiment in a few days, it is not necessary for 20 % ethanol wash. Only DW wash is OK.

Change and wash a sample loop

  1. Change to L (Load) mode. Set flow-path to 1-1.
  2. Install a loop.
  3. Wash the loop (Ethanol -> DW -> Buffer)
    1. Change to I (Inject) mode.
    2. Draw buffer into 20 mL syringe. Push out bubbles.
    3. Install a needle to the syringe.
    4. Insert the needle to the injection port
    5. Change to L (Load) mode.
    6. Push the syringe to fill the sample loop with buffer
    7. Change to I (Inject) mode.
    8. Pull out the needle.

Sample preparation

Before starting the experiments, set the appropriate max pressure and test manually by flowing buffer to the column. Check your run protocol.

  1. Set flow-path to 1-1.
  2. Change to I (Inject) mode.
  3. Filtrate your sample with 0.22 µm filter or centrifuge with TLA100.3 rotor 4 ºC, 50,000 rpm, 15 min.
  4. Draw buffer into a syringe.
  5. Install a needle and push out buffer, remaining ~ 50 µL.
  6. Draw your sample into the syringe.
  7. Insert the needle to the injection port.
  8. Switch to L (Load).
  9. Push your sample slowly to the loop to avoid bubbles.　Load the sample until it exits the waste tube. Do NOT leave bubbles in the sample loop. This step is irreversible. Handle carefully.
  10. Switch the path to the size exclusion column.
  11. Set collection tubes.
  12. Turn the UV detector on.
  13. Run the program.
  14. Check the drop if it's safely going into collection tubes. The position of the fraction collector is a little off for unknown reason.
  15. After the experiment, turn off the UV detector.
  16. Run a wash program.
